Blackscreen

Instructions:

Gameplay

This is a sound based game that uses key presses to progress. Can be 1 player or 2 player.

Controls

When booting up the game, press [Enter] to start the game.
Player 1 uses WASD, Player 2 uses arrow keys. Up to jump and down to pipe.
Collect coins using [C].
The game ends when pressing [X].

Creation Method

This activity was created using HTML audio elements and Javascript, all in the frontend.
