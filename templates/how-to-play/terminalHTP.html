Terminal - How to play

Gameplay

This is a single-player experience, similar to a puzzle game.
Navigate the virtual environment using commands, and learn the secrets of the terminal.
Even if you are familiar in an existing operating system, this activity is still guaranteed to be a brand new experience.
Try to reach the end of the game!

Estimated Playtime: 30 minutes - 1 hour

This activity was made using python, mostly backend code.

Controls

Just type in the terminal input box.
